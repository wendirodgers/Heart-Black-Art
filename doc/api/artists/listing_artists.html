Artists API

Fetch artists from the database based on optional parameters

Listing artists

GET /artists

Parameters

Name Description
media List of media that the artists use in their work.
instagram Artists in results must have an Instagram account.
name Results should return name of artist (or Artists in results must have a name).
location List of locations where an artist is domiciled.
gender Artists in results should return a gender.
age Artists in results must have an age range.
facebook Artists in results must have a Facebook account.
tumblr Artists in results must have a Tumblr account.
twitter Artists in results must have a Twitter account.

Request

Headers

Host: example.org
Cookie:

Route

GET /artists?media[]=Acrylic&media[]=Photography&instagram=true&name[]=Jojo+Smith&location[]=Austin%2C+TX&location[]=New+York+City&gender[]=male&gender[]=female&age[]=25+and+under&facebook=true&tumblr=true&twitter=true
